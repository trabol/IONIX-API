

## Archivo de configuracion
Cambiar el nombre del archivo env.text a .env

## instalar dependencias

```bash
$ npm install
```

## Running the app local

```bash
# watch mode
$ npm run start:dev

```

## Test

```bash
# unit tests
$ npm run test

```

## Running from docker-compose

```bash
# run docker
$ docker-compose up --build

```
